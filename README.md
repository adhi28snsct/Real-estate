# EstateFlow – Real Estate Lead Tracker (Demo Version)

EstateFlow is a modern SaaS-style lead tracking system built specifically for real estate agencies.

This demo version is designed for client presentations and validation before full backend implementation.

---

## 🚀 Purpose

Real estate agencies receive inquiries from:
- WhatsApp
- Website forms
- Property portals
- Direct calls

Most agencies track leads manually using Excel or chat history.

EstateFlow provides a simple dashboard to:
- Track property inquiries
- Update lead status
- Monitor follow-ups
- View conversion performance

---

## 🧱 Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Frontend-only architecture (mock data)
- No backend / No database (demo version)

---

## 🎯 Features

### Dashboard
- Total Leads
- New Leads
- Follow Ups
- Closed Deals
- Conversion Rate %

### Leads Management
- Add new property inquiry
- Update inquiry status
- Filter by status
- Realistic UAE-based sample data
